# 水下机器人 - PC 端通信与监控系统

## 项目概述
水下机器人 PC 端控制与监控系统，通过以太网实现 PC ←→ STM32 双向通信。

## 硬件配置
- **MCU**: STM32（已烧录以太网驱动）
- **通信方式**: 以太网模块（TCP 协议）
- **电脑 IP**: `192.168.1.100`
- **STM32 IP**: `192.168.1.55`
- **通信端口**: `8888`（PC 监听，STM32 主动连接）

## 通信规范
- **协议**: TCP
- **方向**: STM32 → PC（持续发送），PC → STM32（网页发送指令）
- **消息格式**: 纯文本
- **消息分隔**: 支持 `\r\n`、`\n\r`、`\n`、`\r`
- **发送节奏**: STM32 持续发送，约 1 条/秒
- **粘包处理**: 按换行符分割消息

## 系统架构
```
┌──────────────┐   WebSocket   ┌──────────────────┐   TCP(主动连)  ┌──────────────┐
│ 网页前端      │ ◄──────────► │ Node.js 中转服务  │ ◄───────────  │ STM32 + 以太网│
│ (浏览器)      │              │ (localhost:8080)  │               │ 192.168.1.55 │
└──────────────┘              └──────────────────┘               │  → :8888     │
                                                                    └──────────────┘
```

## 中转服务说明
- PC 作为 TCP 服务端，在 `0.0.0.0:8888` 监听，等待 STM32 连接
- 使用 Node.js 内置 `net` 模块建立 TCP 服务
- 手写 WebSocket 协议（零外部依赖），供网页前端连接
- 双向转发：STM32 → TCP → WebSocket → 网页；网页 → WebSocket → TCP → STM32
- 使用 `http` 模块提供静态网页服务
