# 水下机器人 PC 端通信与监控系统

通过以太网实现 PC ↔ STM32 双向通信的水下机器人控制与监控系统。

## 系统架构

\\\
┌──────────────┐   WebSocket   ┌──────────────────┐   TCP(主动连)  ┌──────────────┐
│ 网页前端      │ ◄──────────► │ Node.js 中转服务  │ ◄───────────  │ STM32 + 以太网│
│ (浏览器)      │              │ (localhost:8080)  │               │ 192.168.1.55 │
└──────────────┘              └──────────────────┘               │  → :8888     │
                                                                    └──────────────┘
\\\

## 硬件配置

| 项 | 值 |
|---|---|
| **MCU** | STM32（已烧录以太网驱动） |
| **通信方式** | 以太网模块（TCP 协议） |
| **电脑 IP** | \192.168.1.100\ |
| **STM32 IP** | \192.168.1.55\ |
| **通信端口** | \8888\ |

## 通信规范

- **协议**: TCP
- **方向**: STM32 → PC（持续发送），PC → STM32（网页发送指令）
- **消息格式**: 纯文本，支持 \\\r\\n\、\\\n\\r\、\\\n\、\\\r\ 分隔
- **发送节奏**: STM32 持续发送，约 1 条/秒
- **粘包处理**: 按换行符分割消息

## 快速开始

### 前置条件

- Node.js（任意版本）

### 启动

\\\powershell
cd server
node index.js
\\\

浏览器打开 \http://localhost:8080\

### 一键启动

双击或运行根目录下的 \start.ps1\。
